## 주문 생성

***

1. 업체 담당자

- 자신이 소속된 업체의 주문만 생성할 수 있다.
- 요청 업체 ID(receiverCompanyId)를 본인의 companyId로 고정한다.
- 도착 허브 ID는 해당 업체가 소속된 기본 허브 ID로 배정됨


2. 허브 관리자

- 자신이 담당하는 허브를 기점으로 발생하는 주문만 생성
- 도착 허브 ID를 본인 담당 hubId로 고정
- 선택한 요청 업체가 실제 자신이 관리하는 허브 소속된 업체인지 검증

3. 배송 담당자

- 물류 현장에서 긴급 대리 주문을 넣는 시스템, 허브 관리자와 유사
- 본인 소속 허브 기준 도착 허브가 검증되어야 한다.

4. 마스터 관리자
- 제약 사항 없음
